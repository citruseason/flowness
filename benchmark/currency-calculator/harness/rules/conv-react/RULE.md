# React Conventions

## Overview
React 컴포넌트 및 훅 작성 시 따라야 할 네이밍 및 스타일 컨벤션.

## When to Apply
- React 컴포넌트, 훅, 이벤트 핸들러를 작성할 때
- JSX/TSX 파일을 리뷰할 때

## Rules

| Rule | Impact | Description |
|------|--------|-------------|
| [Handler Naming](./handler-naming.md) | MEDIUM | Props는 onX, 핸들러 함수는 handleX |
